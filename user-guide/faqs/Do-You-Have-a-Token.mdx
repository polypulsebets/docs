# Native Token

PolyPulseBets operates using **USDC** for betting and **PLS** for gas.

We do **not** currently have a governance token or airdrop program. Be wary of scams claiming to sell "$PPB" tokens. Any official announcements regarding a token will come solely from our official X (Twitter) or Discord.
